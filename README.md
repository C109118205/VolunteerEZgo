# VolunteerEZgo 志工EZgo
## 架構
前端：Vue <br>後端：Python-FastAPI <br>資料庫：MariaDB 
## started

### frontend
```
/frontend/
npm install
```
### Backend
```
/Backend/
pip install -r requirement.txt
```

## run test

### Vue
```
/frontend/login
npm run dev
```

### FastAPI
```
/Backend/src
uvicorn main:app --reload
```
